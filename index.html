OSHIHUB

複数の推し(VTuber)を、ひとつのアプリで。

OshiHubは、複数のVTuberを応援するファンのためのiOSアプリです。 登録したVTuberの配信状況・最新動画をまとめて確認でき、 YouTube / Twitchをまたいだ視聴、応援リンクの整理などをサポートします。

🚧 現在開発中（個人開発・未リリース）

主な機能

複数VTuberの登録・配信通知、マルチビュー視聴、埋め込みライブチャット、応援リンク（X / BOOTH / マシュマロ）の整理

使用しているAPI

YouTube Data API v3（チャンネル情報・配信状況・最新動画の取得）を利用しています
