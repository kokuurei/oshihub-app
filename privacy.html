プライバシーポリシー

最終更新日: 2026年7月17日

OshiHub（以下「本アプリ」）は、個人開発者が開発するiOSアプリです。本ポリシーは、本アプリの情報の取り扱いについて説明します。

取得する情報

本アプリは、ユーザーの氏名・メールアドレス・位置情報などの個人情報を収集しません。ユーザーが登録したVTuberのチャンネル情報（表示用のチャンネル名・アイコン・登録者数・動画情報など）は、YouTube Data API v3を通じて取得し、お使いの端末内にのみ保存されます。

外部サーバーへの送信

本アプリは自社サーバーを保有しておらず、取得した情報を外部サーバーへ送信することはありません。すべてのデータは端末内（UserDefaults）に保存されます。

外部サービスの利用

  • YouTube Data API v3 / IFrame Player API: チャンネル情報の取得、動画の埋め込み再生に使用しています。YouTubeの利用についてはGoogleのプライバシーポリシーもあわせてご確認ください。
  • Twitch Embed API: Twitchチャンネルの埋め込み再生に使用しています。
